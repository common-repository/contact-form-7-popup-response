Contact Form 7 Popup Response helps you to display Contact Form 7 responses or validation messages in a popup

== Description ==

This plugin displays Contact Form 7 messages in popup. You can display;

* Validation errors
* Successfull validations
* Response

messages in popup

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `contact-form-7-popup-response` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
